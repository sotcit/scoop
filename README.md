# scoop

## 添加仓库

```shell
scoop bucket add scoop https://github.com/sotcit/scoop.git
```

## 安装软件

baidupcs-go

```shell
scoop install scoop/baidupcs-go
```

aliyunpan

```shell
scoop install scoop/aliyunpan
```

365

```shell
scoop install scoop/365
```

he3

```shell
scoop install scoop/he3
```

lx-music

```shell
scoop install scoop/lx-music
```
